Atributos
Enlaces
Se utiliza para crear enlaces a otros documentos o recursos, ya sea dentro del mismo sitio web (internos) o en otros sitios web (externos)
< a (atributo)...< /a>
href, target(_self o _blank), title, download, rel(nofollow o noopener noreferrer), y type
Imagenes Permite insertar imágenes en una página web < img(atributos)> src(URL de la imagen que se desea mostrar), width, height y title
Audio Se utiliza para reproducir audio en una página web < audio(atributos)>...< /audio> src, controls, autoplay, loop, y type
Video Se utiliza para insertar videos en una página web. Permite especificarla fuente del video, así como configurar opciones como el tamaño, el formato ... < video (atributos)>...< /video> src, controls, autoplay, loop, width, height, muted, poster, preload, playsinline, y type
Mapas de clicks La etiqueta < map> permite definir zonas de una imagen que pueden ser enlazadas aotras páginas web o a otras partes de la misma página < map (atributo)>...< /map> name(especifica un nombre para el mapa, que se utilizará para vincularlo con una etiqueta < img>)
Mapas de clicks La etiqueta < map> debe tener un conjunto de etiquetas < area>, las cuales definen las zonas de la imagen que serán vinculadas < map> < area (atributos)>...< /map> Shape, coords, href y target(_blank o _self)
Mapas de clicks:
atributo coords		 Define las coordenadas del mapa - rect: el orden en que se define el rectángulo es "izquierda-x, arriba-y, derecha-x, abajo-y", circle: el centro y el radio "centro-x, centro-y, radio"poly: una a una, las coordenadas de los vértices "x1, y1, x2, y2, ..., xn, yn"
Marcos
Se utiliza para insertar contenido de otra página web dentro de una página existente
< iframe (atributos)>...< /iframe>
Src(obligatorio), width, heigth, name, sandbox, seamless, scrolling, srcdoc, frameborder y allowfullscreen
