Tablas
Se utiliza para crear tablas en un documento HTML. Dentro de una tabla, se pueden incluir filas y celdas para organizar y presentar información de manera estructurada
< table>...< table>
Border, width, align, cellpadding, cellspacing, bgcolor, frame y rules
Filas Representa una fila en una tabla. Se utiliza dentro de una etiqueta < table> para definir una fila de celdas < table> < tr>...< /tr> < /table> Align y bgcolor
Celdas Es utilizada dentro de una fila (etiqueta < tr>) para crear una celda de datos. Puede contener cualquier tipo de contenido < table> < tr> < td>...< /td> < /tr> < /table> Colspan, rowspan y align
Encabezados Se utiliza para definir un encabezado de celda en una tabla < th>...< /th> Colspan, rowspan, scope y align
Combinar celdas por filas El atributo colspan extiende la celda tantas celdas a la derecha como le indiquemos,contando la celda actual colspan -
Combinar celdas por columnas El atributo rowspan extiende la celda tantas celdas hacia abajo como le indiquemos,contando la celda actual rowspan -
Anidar tablas
Dentro de cualquier etiqueta < td> se puede anidar otra tabla. Simplemente añade las marcas de la nueva tabla entre < td> y < /td>
