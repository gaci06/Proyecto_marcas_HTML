Contenedor del formulario
Se utiliza para crear un formulario en una página web
< form>...< /form>
id, action, method, autocomplete, enctype, novalidate, target
Controles de entrada de datos Permite a los desarrolladores recopilar información del usuario a través de diversos tipos de campos de entrada < input (atributos)> id, name, value, placeholder, required, readonly, disable, type
Tipos de elementos < input> Para crear un campo de entrada de texto simple, se utiliza el tipo "text" El tipo "password" crea un campo de entrada donde los caracteres ingresados se ocultan Las casillas de verificación permiten a los usuarios seleccionar múltiples opciones de una lista. El tipo es “checkbox”
Tipos de elementos < input> Los botones de opción, tipo “checkbox”, permiten seleccionar una única opción de un conjunto El tipo "number" crea un campo de entrada para ingresar valores numéricos. Puedes especificar el rango de números permitidos utilizando los atributos min y max El tipo "email" crea un campo de entrada específico para direcciones de correo electrónico
Tipos de elementos < input> El tipo "hidden" crea un campo invisible para el usuario, pero que se manda al servidor El tipo "file" permite a los usuarios cargar archivos. Los usuarios pueden seleccionar un archivo de su dispositivo utilizando el cuadro de diálogo de carga de archivos El tipo "submit" crea un botón que envía el formulario y sus datos al servidor
Tipos de elementos < input> El tipo "reset" un botón que resetea el formulario El tipo "button" crea un botón genérico que puede usarse para diversas acciones en la página, como abrir una ventana emergente, enviar datos mediante JavaScript, etc El tipo "range" crea un control deslizante que permite a los usuarios seleccionar un valor dentro de un rango específico
Tipos de elementos < input> El tipo "tel" crea un campo de entrada para números de teléfono El tipo "color" crea un botón de selección de un color a través de una paleta de colores El tipo "image" crea un control como el submit, pero usando una imagen
Tipos de elementos < input> El tipo "url" crea un campo de entrada para URL El tipo "date" crea un campo de entrada para seleccionar fechas utilizando un selector de fecha nativo del navegador El tipo "datetime-local" crea un campo de entrada para seleccionar fecha y hora
Tipos de elementos < input> El tipo "month" crea un campo de entrada para seleccionar un mes y de un año El tipo "week" crea un campo de entrada para seleccionar una semana de un año -
Etiquetas para controles Se utiliza para asociar una descripción con un elemento de formulario específico < label (atributos)>...< /label> for y form
Áreas de texto Se utiliza para crear un área de texto multilínea en un formulario web < textarea (atributos))>...< /textarea> Autofocus, cols, disabled, form, maxlength, name, placeholder, readonly, required, rows, wrap
Listas de selección Se utiliza para crear un menú desplegable < select (atributos)>...< /select> Autofocus, disabled, form, multiple, name, required, size
Elementos de listas de selección Representa una opción en una lista desplegable.Cada opción tiene un valor asociado que se envía al servidor cuando se selecciona la opción en el formulario < option (atributos)>...< /option> Value, label, selected, disabled
Grupos en listas de selección Se usa para agrupar opciones relacionadas dentro de una lista desplegable(elemento < select>) < optgroup>...< /optgroup> label y disabled
Listas de selección Se utiliza para crear una lista de opciones predefinidas que se pueden utilizar en un campo de entrada de texto < datalist>...< /datalist> id
Botones Se utiliza para crear un botón interactivo en una página < button>...< /button> type, autofocus, disabled, form, name
Agrupación de elementos La etiqueta < fieldset> se utiliza para agrupar elementos relacionados en un formulario web.El elemento < fieldset> se utiliza junto con la etiqueta < legend> para proporcionar un título para el grupo de elementos < fieldset> < legend>...< /legend> < /fieldset> disabled, form, name
Salida de datos Se utiliza para representar el resultado de un cálculo o una operación realizada por uns cript en una página web < output>...< /output> for, form, name, value
Avance de procesos
Se utiliza para mostrar el progreso de una tarea
< progress(atributos)>...< /progress>
Max, form, name, value
